Character Strings

  • Character Strings: Sequences of characters
  • Design Considerations:
    • Primitive or special array?
    • Static or dynamic length?

  • Operations:
    • Assignment (==)
    • Catenation
    • Substring Reference
    • Comparison (=, >, ...)
    • Pattern Matching
  • Type Evaluation:
    • Aids writability
    • Including strings as primitives is not costly in either language or compiler complexity
    • Dynamic strings are more flexible but overhead should be considered
    • String operations are essential and should be included
  • Implementation:
    • Static - compile-time descriptor
    • Limited Dynamic Length: may need run-time descriptor
    • Dynamic Length: run-time descriptor (alloc/dealloc is biggest problem)
