Associative Arrays

  • Associative Array: an unordered collection of data elements that are indexed by an equal number of values called keys
    • User-defined keys must be stored in the structure
    • Directly supported by Perl, Python, Ruby and Lua
    • Supported by the Standard Class Libraries in Java, C++, and C#
  • Design Issues:
    • What is the form of references to their elements?
  • Examples using Perl: associative arrays are called hashes
    • Hashes can be set to literal values with the assignment statement
      • %salaries = ("Gary" => 75000, "Perry" => 57000, "Mary" => 55750, "Cedric" => 47850);
    • The value part of hash elements is a scalar, so references to hash element values use scalar names
      • $salaries {"Perry"} = 58850; //also used to add elements
    • Remove elements with the delete operator
      • delete $salaries{"Gary"};
